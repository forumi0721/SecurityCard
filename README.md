# 보안카드 (SecurityCard)

안전하고 간편하게 은행 보안카드와 계좌 정보를 관리할 수 있는 안드로이드 애플리케이션입니다. 

## 🌟 주요 기능

- **보안카드 관리:** 실물 보안카드의 코드표를 앱 내에 안전하게 저장하고 조회할 수 있습니다.
- **계좌 정보 통합 관리:** 카드별로 관련된 은행 계좌번호와 예금주 정보를 함께 등록하고, 원터치로 계좌번호를 복사할 수 있습니다.
- **스마트 검색:** 이체 시 필요한 보안카드 2자리 위치를 입력하면 해당하는 코드를 즉시 찾아주는 기능을 제공합니다.
- **안전한 데이터 보관:** 기기 내부에 암호화(AES)되어 안전하게 저장되며, 외부 서버로 데이터를 전송하지 않습니다.
- **데이터 백업 및 복원:** 등록된 모든 데이터를 JSON 파일로 내보내거나(Export), 다른 기기로 가져올(Import) 수 있습니다.
- **편리한 UI/UX:** 필요한 코드를 터치하여 바로 복사할 수 있으며, 카드 목록의 순서를 자유롭게 정렬할 수 있습니다.

## 🛠 사용된 기술

- **UI:** Jetpack Compose, Material Design 3
- **Architecture:** MVVM (Model-View-ViewModel)
- **Database:** Room (로컬 DB)
- **Security:** Android Keystore 시스템을 이용한 AES-GCM 암호화 (CryptoManager)
- **DI:** Dagger-Hilt
- **Navigation:** Jetpack Navigation Compose

## 🚀 설치 및 실행 방법

1. GitHub 저장소에서 프로젝트를 클론합니다.
```bash
git clone https://github.com/forumi0721/SecurityCard.git
```
2. Android Studio(최신 버전 권장)에서 프로젝트를 엽니다.
3. 안드로이드 기기를 연결하고 앱을 빌드 및 실행합니다.

## 📱 주요 화면 안내

- **메인 화면:** 등록된 카드 목록을 확인하고 우측 상단 메뉴(⋮)를 통해 카드를 추가하거나 목록 순서를 정렬할 수 있습니다.
- **조회 화면:** 
  - **검색 모드:** 찾고자 하는 코드의 위치를 입력하면 결과를 보여줍니다.
  - **전체 보기:** 보안카드 전체 코드를 한눈에 확인할 수 있습니다.
  - 화면 상단의 **계좌 정보**를 누르면 계좌번호가 복사되며, **코드 번호**를 터치해도 즉시 클립보드에 복사됩니다.
- **등록/수정 화면:** 가로/세로 방향, 행/열 개수에 맞춰 보안카드 코드를 일괄 입력하고 계좌 정보도 다중으로 등록할 수 있습니다.

## ⚠️ 주의 사항

- 본 앱은 오프라인 환경을 기반으로 동작합니다. 앱 삭제 또는 기기 변경 전에 **반드시 설정 메뉴에서 데이터 내보내기(Export)를 통해 백업**을 진행하시기 바랍니다.
- 루팅된 기기 등 OS 수준의 보안이 취약한 환경에서의 사용은 권장하지 않습니다.
